# utenti-flask
Questo progetto è una semplice API REST realizzata in Python con Flask che permette la gestione di utenti.
L’applicazione implementa operazioni CRUD (Create, Read, Update, Delete) su una collezione di utenti memorizzata in memoria e genera dati realistici utilizzando la libreria Faker.

L’API espone endpoint per:

- recuperare tutti gli utenti

- recuperare un utente per ID

- creare nuovi utenti

- aggiornare utenti esistenti

- eliminare utenti

- eliminare tutti gli utenti

Il server è configurato per l’esecuzione locale e rappresenta un esempio didattico di backend REST in Flask.
